Medical Insurance Cost Prediction

## TechCrush Capstone Project

### Team Name
CareCast-Team

---

## Project Overview

CareCast is a machine learning project that predicts an individual's medical insurance costs based on demographic and lifestyle factors. The project aims to help insurance providers and customers estimate healthcare expenses using predictive analytics.

---

## Problem Statement

Medical insurance costs vary significantly due to factors such as age, BMI, smoking habits, and family size. Estimating these costs manually is difficult and often inaccurate.

This project develops a predictive model capable of estimating insurance charges from customer information.

---

## Objectives

- Clean and preprocess the dataset.
- Perform Exploratory Data Analysis (EDA).
- Identify factors influencing insurance costs.
- Build and compare machine learning models.
- Evaluate model performance.
- Present actionable business insights.

---
